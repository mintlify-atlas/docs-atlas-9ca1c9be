## What is D&D Textil?

D&D Textil is a full-stack digital textile store built for SENA's ADSO program. It provides a complete e-commerce experience for buying and selling textile fabrics, with dedicated portals for customers, sellers, and administrators.

The platform combines a **React 18** frontend with a **Java backend** connected to a **PostgreSQL** database, offering a real-world web application architecture with role-based access control.

## User roles

D&D Textil has three distinct user roles, each with different access levels:

| Role | Spanish | Access |
|---|---|---|
| `client` | Cliente | Shopping, order history, wishlist, support tickets |
| `seller` | Vendedor | Product management, order processing, stock alerts |
| `admin` | Administrador | Full platform access including all seller + analytics + user management |

<Note>
The admin role inherits all seller capabilities. Routes protected with `roles={['seller', 'admin']}` are accessible by both.
</Note>
